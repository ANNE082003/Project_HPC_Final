Software and Applications

Computer Center CO Installed Software

There are large numbers of packages that are helpful to all users, but not available from the base system. For these types of packages, CC will compile and install them and make them available via "modules" which can be loaded to add them to your environment. These include things like MPI, compilers, libraries, and other applications. If there is something that you think should be made available to all users in this manner, email cc@tifr.res.in and we can see about installing it for you. To see what is already installed, you can run "module avail" to get a current list.

List of Applications

  • NAMD is computer software for molecular dynamics simulation.
  • ROOT analyzing petabytes of data, scientifically. An open-source data analysis framework used by high energy physics and others.
  • MATLAB: Math Graphics Programming. Whether you're analyzing data, developing algorithms, or creating models. MATLAB is designed for the way you think and the work you do.
